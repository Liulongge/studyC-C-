## 语言

## 有限状态机四大要素
    现态：当前所处状态
    次态：当条件满足后，即将转移的下一个状态
    动作：当满足某个事件时执行的动作；动作执行完毕后可以转移到另一个状态或保持原有状态
    条件：转移状态所需的条件，当满足条件时，会触发一个动作或进行状态转移
